# Local SEO Planner

## Context

This project is **Show Me Home Ventures**, a real estate / home-buying business serving Springfield MO and surrounding towns. Every page structure decision must reinforce local search authority.

## Core Service Area

- **Primary**: Springfield, MO
- **Secondary towns** (update as the business expands):
  Ozark, Nixa, Republic, Battlefield, Willard, Rogersville, Strafford, Marshfield, Bolivar, Branson, Joplin, Lebanon, Aurora, Monett, Carthage, Mt. Vernon, Ash Grove, Fair Grove, Clever

## Step 1: Keyword Cluster Research

When the user asks for keyword research or page planning:

1. **Group keywords into clusters** around buyer intent:
   - Service clusters: "sell my house fast," "cash home buyers," "we buy houses," "sell inherited house," "sell house as-is"
   - Location clusters: each town gets its own cluster modifier (e.g., "sell my house fast Ozark MO")
   - "Near me" intent: "home buyers near me," "sell house near me," "cash for houses near me"
   - Problem clusters: fire damage, foreclosure, divorce, probate, inherited, tenant issues, tax liens, code violations

2. **Output format** for each cluster:

```
Cluster: [name]
Primary KW: [highest-volume keyword]
Supporting KWs: [3-8 related terms]
Intent: informational | transactional | navigational
Recommended page type: service | location | blog | landing
```

## Step 2: Page Map / Sitemap

Generate a hierarchical sitemap covering:

```
/                             (homepage — primary city + brand)
/about/
/how-it-works/
/testimonials/
/contact/

# Service pages (one per core offer)
/sell-my-house-fast/
/we-buy-houses-springfield-mo/
/cash-home-buyers/
/sell-house-as-is/
/sell-inherited-house/
/foreclosure-help/
/[additional-service-slug]/

# Location pages (one per town)
/locations/springfield-mo/
/locations/ozark-mo/
/locations/nixa-mo/
/locations/[town-slug]/

# Blog / resource hub
/blog/
/blog/[post-slug]/
```

Rules:
- Every service page targets `[service keyword] + Springfield MO`.
- Every location page targets `[core offer] + [town]` and links to the parent service page.
- Location pages must include: local proof (address mention, map embed placeholder), a unique intro paragraph, and a CTA.

## Step 3: Internal Linking Plan

Produce a linking matrix:

| Source page | Target page | Anchor text pattern |
|-------------|-------------|---------------------|
| /sell-my-house-fast/ | /locations/ozark-mo/ | "sell your house fast in Ozark" |
| /locations/ozark-mo/ | /sell-my-house-fast/ | "our cash home-buying process" |
| /blog/inherited-house-guide/ | /sell-inherited-house/ | "learn how we buy inherited houses" |
| Homepage | each service page | service keyword anchor |
| Each location page | 2-3 nearest-town location pages | "also serving [town]" |

Rules:
- Every page should have 3-5 internal links minimum.
- Use keyword-rich but natural anchor text (no "click here").
- Blog posts always link to the most relevant service page.
- Location pages cross-link to nearby towns and the parent service page.

## Step 4: Title / Meta Templates

### Service pages

```
Title: [Primary KW] | [Brand] — Springfield, MO
Meta: [Brand] provides [service] in Springfield MO & surrounding areas. Get a fair cash offer today — no repairs, no fees. Call [phone].
H1: [Primary KW] in Springfield, MO
```

### Location pages

```
Title: [Core offer] in [Town], MO | [Brand]
Meta: Looking to [action] in [Town], MO? [Brand] buys houses for cash — any condition. Get your free offer in 24 hours.
H1: [Core offer] in [Town], MO
```

### Blog posts

```
Title: [Long-tail keyword] — [Brand] Blog
Meta: [1-2 sentence summary answering the searcher's question]. Learn more from Springfield's trusted home buyer.
H1: [Long-tail keyword phrased as a question or how-to]
```

## Output Checklist

When delivering a local SEO plan, include:

- [ ] Keyword clusters (minimum 5 clusters)
- [ ] Full sitemap with URL slugs
- [ ] Internal linking matrix
- [ ] Title + meta + H1 for every page
- [ ] "Near me" intent handling note
- [ ] Secondary-town expansion roadmap
